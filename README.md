
# Donation Campain


## Features

- When users want to make a donation, clicking on a donation stick will take them to the details cart where they can find all the information about the donation and a button with the  amount.Donation will be completed by pressing the button.

- There is a button on navbar name donation when we click in that button you will see the all details of your donation

- Another page is Statistics by click the statistic button in navbar you will see the donation statistics how many donation you have the opportunity and how many you donate.
